## How It Works

<Steps>
  <Step title="Get Your Free API Key">
    Visit [Google AI Studio](https://aistudio.google.com/app/apikey) and generate a free Gemini API key. The free tier includes 1500 requests per day.
  </Step>

  <Step title="Upload Your Answer">
    Upload a PDF or multiple images (JPG/PNG/WebP) containing your question and answer. The AI automatically extracts both using OCR.
  </Step>

  <Step title="Select Subject & Marks">
    Choose from:
    - **GS1** (10/15 marks) — Art & Culture, History, Society, Geography
    - **GS2** (10/15 marks) — Polity, Governance, Social Justice, International Relations
    - **GS3** (10/15 marks) — Economy, Science & Tech, Environment, Internal Security
    - **GS4** (10/20 marks) — Ethics, Integrity, Aptitude
    - **Essay** (125 marks) — 4-section framework evaluation
  </Step>

  <Step title="Get Detailed Feedback">
    Receive a comprehensive evaluation with:
    - Overall score and percentage
    - Category-wise breakdown (4 categories)
    - Key points covered vs missing
    - Improvement tips (5 specific suggestions)
    - Model answer structure
    - Strengths analysis
  </Step>
</Steps>

## What Makes It Special?

### Subject-Specific Rubrics

Each subject has dedicated evaluation criteria:

- **GS1**: PESTEL+ multidimensional analysis with maps/diagrams emphasis
- **GS2**: IC5 structure (Introduction → Context → Current Relevance → Conclusion) with constitutional anchoring
- **GS3**: Data-driven frameworks with policy/institutional references
- **GS4**: Behavioral ethics with administrative pragmatism
- **Essay**: 4-section framework (Structure, PESTEL+, Thematic Depth, Expression)

### Topper Benchmarking

Answers are evaluated against real UPSC topper standards:

- Must mention specific **Articles** (14, 15, 21, 38, 51A, etc.)
- Reference **Committees** (2nd ARC, Sachar, Mandal, Kasturirangan)
- Cite **Case Laws** (Puttaswamy, Sabrimala, Vishaka, Bommai)
- Include **Data** (reports, statistics, frameworks)
- Use **Technical Terms** specific to each subject

### Intelligent OCR

The AI automatically:

- Identifies the question (even if on any image, not just the first)
- Determines the correct reading order for multi-page answers
- Preserves structure (subheadings, bullet points, numbered lists)
- Detects diagrams, tables, and maps with inline tags
- Handles both printed questions and handwritten answers

<Note>
  **Privacy First**: Your API key is stored only in your browser's localStorage. All processing happens client-side. We never see your answers or API key.
</Note>
