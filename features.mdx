## 2. Intelligent OCR & Content Extraction

### Multi-Format Upload Support

The evaluator handles various input formats seamlessly:

<Steps>
  <Step title="PDF Upload">
    **Automatic Text Extraction**

    ```typescript
    // From PdfUpload.tsx:86 - PDF.js extraction
    const extractTextFromPdf = async (pdfFile: File): Promise<string> => {
      const arrayBuffer = await pdfFile.arrayBuffer();
      const pdf = await pdfjsLib.getDocument({ data: arrayBuffer }).promise;
      
      for (let i = 1; i <= pdf.numPages; i++) {
        const page = await pdf.getPage(i);
        const textContent = await page.getTextContent();
        const pageText = textContent.items.map(item => item.str).join(' ');
        fullText += pageText + '\n\n';
      }
    };
    ```

## 4. Strict UPSC Evaluation Standards

### 70% Scoring Ceiling

The evaluator enforces a **strict 70% maximum** to match real UPSC standards:

```typescript
// From gemini.ts:32 - Ceiling enforcement
const ceiling = Math.floor(params.marks * 0.7);

"STRICT CEILING: Never exceed 7/10 or 10.5/15.
In UPSC, 60-65% is a Rank-1 score."
```

<Warning>
  **Why 70%?** In actual UPSC, toppers score 60-65% on average. A score of 7/10 or 10.5/15 is exceptionally strong. The evaluator prevents inflated scoring to keep you realistic.
</Warning>

### Generic Answer Penalty

Answers without specific technical content are heavily penalized:

```typescript
// From gs1Rubrics.ts - Generic penalty
"GENERIC PENALTY: If answer is correct but lacks Non-Negotiables 
(Data/Maps/Terms), max score is 4/10."
```

**Non-Negotiables include**:
- Specific articles, cases, committees
- Data points and statistics
- Technical terminology
- Visual aids (diagrams, maps, tables)

```typescript
export const downloadPdf = async ({
```

## 7. Performance Optimizations

### Web Workers for Non-Blocking Evaluation

```typescript
// From gemini.ts:514 - Worker-based evaluation
if (typeof Worker !== 'undefined') {
  const worker = new Worker(
    new URL('../workers/evaluationWorker.ts', import.meta.url),
    { type: 'module' }
  );
  
  worker.postMessage({ apiKey, prompt, marks });
  
  worker.onmessage = (e) => {
    if (e.data.type === 'result') {
      resolve(e.data.result);
    }
  };
}
```

**Benefits**:
- UI remains responsive during evaluation
- No freezing or lag
- Automatic fallback to main thread if worker fails
- 2-minute timeout for long evaluations

## 9. Dark Mode

Toggle between light and dark themes with persistent storage:

```typescript
// From Index.tsx:48 - Dark mode implementation
const [isDark, setIsDark] = useState(() => {
  if (typeof window !== 'undefined') {
    return localStorage.getItem('theme') === 'dark';
  }
  return false;
});

useEffect(() => {
  document.documentElement.classList.toggle('dark', isDark);
  localStorage.setItem('theme', isDark ? 'dark' : 'light');
}, [isDark]);
```

**Features**:
- Instant theme switching
